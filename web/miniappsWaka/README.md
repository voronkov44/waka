
  # Premium Telegram Mini App (Copy)

  This is a code bundle for Premium Telegram Mini App (Copy). The original project is available at https://www.figma.com/design/gjfppWD4AlVDAzB2fo1s6L/Premium-Telegram-Mini-App--Copy-.

  ## Running the code

  Copy `.env.example` to `.env` and set `VITE_API_BASE_URL` to your backend URL.

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
